# Push-T Subgoal Distillation Round 1 Artifacts

This folder contains the finished public artifacts from the `vw2_subgoal_distill` experiment.

## Included

- `subgoal_distill_report.tex` and `subgoal_distill_report.pdf`
- `eval_subgoal_50rollouts_100steps/summary.json`
- BC and TeacherOracle per-episode CSVs
- 10 BC rollout videos
- 10 TeacherOracle rollout videos
- TeacherOracle config, environment snapshot, frozen requirements, and CSV metrics

## Excluded

- training checkpoints
- raw datasets
- local caches

The run hard-stopped after Gate A failed. No student-stage artifacts exist for this round.
