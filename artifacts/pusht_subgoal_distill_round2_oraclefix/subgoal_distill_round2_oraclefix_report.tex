\section*{1. What Was Tested}

Gate A asks whether future information can be compressed into a continuous subgoal that still drives successful Push-T world rollouts through the shared action decoder.

\begin{itemize}
  \item \textbf{TeacherOracle}: encodes a future rollout window into a continuous subgoal.
  \item \textbf{Student}: predicts that subgoal from history; skipped because TeacherOracle failed Gate A.
  \item \textbf{Fix applied}: oracle future windows advance at every rollout step instead of reusing the step-0 oracle subgoal.
\end{itemize}
