# Repository Agent Instructions

- Apply Occam's razor during coding, experimenting, and report writing.
- Prefer the simplest change that preserves correctness, reproducibility, and the evidence chain.
- Keep experiment reports focused on verdict, evaluation scope, key metrics, sanity checks, and artifact paths.
- Move secondary architecture detail to `docs/` and avoid repeating the same explanation across README, report, and artifact notes.
- Remove stale artifacts, redundant prose, and unused branches of analysis when they do not change the decision.
